# Purpose

Dokumen ini digunakan untuk memastikan seluruh deliverable memenuhi standar kualitas sebelum dapat dilanjutkan ke fase berikutnya atau dirilis.

Tujuan:

- Menjamin kualitas deliverable
- Mengurangi risiko defect
- Mendukung governance
- Mendukung release readiness
- Mendukung audit SDLC
