# Verification Criteria

Setiap kebutuhan dianggap tervalidasi apabila:

* Memiliki Business Objective
* Memiliki Functional Requirement
* Memiliki Use Case
* Memiliki Modul Implementasi
* Memiliki Halaman Implementasi
